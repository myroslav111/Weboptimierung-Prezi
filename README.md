# 🌐 Weboptimierung Prezi-Projekt

## 📌 Projektbeschreibung

Dieses Projekt ist eine interaktive Präsentation zum Thema **Weboptimierung**, umgesetzt als „Prezi-ähnliche“ Webanwendung mit **HTML, CSS und Vanilla JavaScript**.

Statt klassischer PowerPoint-Folien wird eine große digitale Leinwand verwendet, auf der sich die Kamera zwischen verschiedenen Themenbereichen bewegt.

---

## 🎯 Ziel des Projekts

Das Ziel dieses Projekts ist es:

- Weboptimierung verständlich und visuell darzustellen
- Performance, UX und SEO interaktiv zu erklären
- zu zeigen, wie moderne Webtechnologien ohne Frameworks eingesetzt werden können
- eine alternative Präsentationsform zu klassischen Slides zu entwickeln

---

## 🧠 Inhalt der Präsentation

Die Präsentation besteht aus fünf Hauptbereichen:

### ❓ Warum Weboptimierung?
- Bedeutung schneller Webseiten
- Auswirkungen auf Nutzerverhalten
- Conversion und Absprungraten

### ⚡ Performance
- Ladezeit-Optimierung
- Bildkomprimierung (WebP)
- Lazy Loading und Caching

### 🎨 UX Design
- Benutzerfreundlichkeit
- klare Navigation
- responsive Layouts

### 🔍 SEO
- Suchmaschinenoptimierung
- semantisches HTML
- Meta-Daten und Struktur

### 🏁 Fazit
- Zusammenspiel von Performance, UX und SEO
- Bedeutung für erfolgreiche Websites

---

## 🧩 Technische Umsetzung

Das Projekt wurde umgesetzt mit:

- **HTML5** für Struktur
- **CSS3** für Layout, Animationen und Styling
- **JavaScript (Vanilla)** für:
  - Kamera-Zoom & Pan-System
  - Navigation zwischen Slides
  - Animationen (z. B. Intro, Typing-Effekte, Icons)
  - Interaktive Facts bei Klick

---

## 🚀 Features

- 🧭 Prezi-ähnliche Kamera-Bewegung (Zoom + Pan)
- 📱 Responsive Design für verschiedene Bildschirmgrößen
- 🎭 Animierte Icons pro Thema
- 💬 Interaktive Facts pro Bullet Point
- ⌨️ Tastatur-Navigation (Pfeiltasten)
- 🖱️ Klickbare Slides
- 🎬 Intro-Zoom (optional Weltall-Effekt)
- 🔊 Sound-Feedback (optional UI-Sounds)

---

## 💡 Lernziel / Nutzen

Dieses Projekt zeigt:

- wie moderne Web-Interfaces ohne Frameworks gebaut werden können
- wie wichtig Performance-Optimierung im Frontend ist
- wie UX und Interaktion die Wahrnehmung einer Anwendung verändern
- wie Präsentationen im Web neu gedacht werden können

---

## 🏁 Fazit

Dieses Projekt verbindet **Webentwicklung + Design + Interaktivität** und demonstriert gleichzeitig die Prinzipien der **Weboptimierung direkt in der Anwendung selbst**.

> „Die Präsentation über Weboptimierung ist selbst ein Beispiel für gute Weboptimierung.“
